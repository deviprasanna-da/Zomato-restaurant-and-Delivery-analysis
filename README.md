# Zomato-Restaurant-and-Delivery-analysis

## Problem Statement
Analyze restaurant data to identify customer preferences, cuisine popularity, restaurant distribution, ratings trends, and online delivery adoption.

---

## Project Objective
- Identify cities with the highest restaurant concentration
- Analyze cuisine popularity using customer engagement
- Evaluate online delivery adoption across cities
- Understand customer rating trends

---

## Tools Used
- Excel
- SQL
- Power BI

---
## Dataset 
Source: Kaggle — zomato Restaurants data-size 9551 restaurant globally, filtered to 8652 India restauranta 

---

## Analysis Performed
- Data Cleaning
- Pivot Table Analysis
- Dashboard Creation
- Customer Engagement Analysis

---

## Key Insights
- New Delhi has the highest restaurant count.
- Online delivery adds only +0.03 stars — delivery ≠ quality but Restaurants with online delivery received       higher votes.
- North Indian cuisine showed the highest customer engagement.
- Metro cities have higher online delivery adoption.
- Only 1.78% of rated restaurants achieve "Excellent" status - Price does predict quality: Budget 3.22★ →
    Luxury 3.74★ (0.52★ gradient)
